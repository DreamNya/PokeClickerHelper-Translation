## 脚本说明

独立脚本
**UI界面**依赖宝可梦点击（Poke Clicker）辅助脚本 脚本核心模块  
（不安装亦可正常运行，但不支持UI界面交互设置及汉化热切换功能）  

采用内核汉化形式，效率较高但缺少兼容性  
目前主要汉化范围：任务线、城镇名称、NPC名称、NPC对话、成就  
（NPC对话由GPT4翻译（\*由ai.telagod.com赞助）、人工粗校对）

未来汉化目标：剩余未汉化内容
